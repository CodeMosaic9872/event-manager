# Client Inputs / Access Needed (From PRD — Event Supplier Marketplace Platform)

One-liner checklist of the access/accounts we need from the client to build the PRD scope (marketplace + AI planning + job board + admin).

## 1) Access / Accounts Needed

- Meeting transcript / notes referenced by the PRD.
- AWS account access (or approval to create/provision project infrastructure).

- PostgreSQL access details for dev/staging/prod (host/port/db/user/SSL) or approval to provision Postgres.

- OpenAI API key + billing owner (and any usage limits).

- Email provider access (or approval to create) + sending domain verification via DNS.
- SMS provider access (or approval to create).
- WhatsApp provider access (only if you want WhatsApp notifications now).

- Firebase project access for Analytics (and access to add our team/service accounts).
